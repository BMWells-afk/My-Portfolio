# CS601 - HW3 - Portfolio

## Project Description
A fun simple porfolio page showcasing my projects
- I added a logo to the header with text effects to the title
- Links to the sections of the page are in the nav header
- Has four project cards in a flex grid format, each with a fake link to said project
- I added a few psuedo-classes to change the hover of the links an the cards
- A sidebar showcases my avatar profile photo with a short bio and email contact link 
- I added a psuedo-element to color the email list item 
- Footer displays a copyright symbol and my name
